# InAmigos Foundation Awareness Webpage

A modern and responsive NGO awareness webpage created for the InAmigos Foundation Internship Task.

## 🌟 Project Overview

This project was developed to spread awareness about the mission, projects, and social impact of InAmigos Foundation.

The webpage highlights:

* NGO introduction
* Ongoing projects
* Social impact statistics
* Gallery section
* Volunteer & donation call-to-action



## 🚀 Features

* Responsive modern design
* Smooth scrolling navigation
* Interactive project cards
* Social impact section
* Gallery with hover effects
* Clean UI using HTML & CSS



## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts
* Font Awesome Icons



## 📌 NGO Projects Included

* Project SEVA
* Project Bachpanshala
* Project Jeev
* Project Udaan
* Project Prakriti
* Project Vikas
